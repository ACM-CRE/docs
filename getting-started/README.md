# Getting Started

This guide will help you set up your ACM COMPUTE Regional Event website.

## What you'll end up with

A website for your event that includes:
- Event schedule
- Speaker and organiser profiles
- Venue information
- Registration (via embedded form)
- Sponsor showcase

## Time required

About 30 minutes for initial setup, then ongoing customisation as you add content.

## Steps

1. **[Prerequisites](prerequisites.md)** - Tools you need to install
2. **Setup** - Choose your operating system:
   - [macOS](macos.md)
   - [Windows](windows.md)
   - [Linux](linux.md)
3. **[Customisation](../customization/README.md)** - Add your event details
4. **[Deployment](../deployment.md)** - Publish your site

## Need help?

Check the [Troubleshooting](../troubleshooting.md) guide or [open an issue](https://github.com/ACM-CRE/acm-cre.github.io/issues) on GitHub.
