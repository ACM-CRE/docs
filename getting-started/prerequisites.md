# Prerequisites

Before setting up your event website, you need a few things installed on your computer.

## Required

### Git

Git is a version control system that tracks changes to your code.

**Check if you have it:**
```bash
git --version
```

If you see a version number, you're good. If not:
- **macOS**: Install Xcode Command Line Tools: `xcode-select --install`
- **Windows**: Download from [git-scm.com](https://git-scm.com/download/win)
- **Linux**: `sudo apt install git` (Ubuntu/Debian) or `sudo dnf install git` (Fedora)

### GitHub Account

You'll need a [GitHub account](https://github.com/signup) to host your website for free using GitHub Pages.

### Text Editor

Any text editor works. We recommend [VS Code](https://code.visualstudio.com/) - it's free and works on all platforms.

## What gets installed during setup

The setup guide will walk you through installing:

- **mise** - A tool that manages programming language versions
- **Ruby** - The programming language Jekyll is built with
- **Bundler** - A tool that installs Ruby packages (called "gems")
- **Jekyll** - The static site generator that builds your website

You don't need to understand these tools deeply. Just follow the setup steps and they'll be installed automatically.
